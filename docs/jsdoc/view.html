<static> this.multiValueAxes :boolean

Allow one or more Axes displayed in chart. Default is false and use the defaultAxesOptions if no axes configuration in collections' renderOptions which had been bound to the view and not hidden. If exist axes configuration in any collection's renderOptions and multiValueAxes is false, we will use the first collection's setting when creating the options.
