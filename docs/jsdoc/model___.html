Module: echarts/model

echarts/model

Backbone model for Baidu Echarts

One model represent for one data in Echarts's series. The categoryField attribute will be summarized and fill to the legend data for representing the category in Pie or xAxis in Line or Bar The valueField attribute will be filled to the series's data.If it's null or undefined, it will be set zero.
